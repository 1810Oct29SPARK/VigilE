Preterhuman Powers:


Telekinesis:
The ability to move objects without touching them.





Object Teleportation:
The ability to send objects from one place to another.





Object Sensing:
The ability to detect other objects.





Object Creation:
The ability to materialize objects out of thin air.





Object Repair:
The ability to fix broken objects.





Object Disintegration:
The ability to de-materialize objects.





Object Form:
The ability to transform all or part of the body into a material(s).





Object Aging:
The ability to change the age of objects.





Object Override:
The ability to control the functions of objects.





Object Merging:
The ability to pass through solid objects.





Object Sizing:
The ability to change the size of objects.





Object Conversion Touch:
The ability to change matter into an object by touching it.
